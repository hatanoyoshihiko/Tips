# Nginx tips

## How to install

- RHEL  
`# dnf install nginx`

Note: dependency packages are also installed  

- nginx-all-modules
- nginx-mod-http-image-filter
- nginx-mod-http-perl
- nginx-mod-http-xslt-filter
- nginx-mod-mail
- nginx-mod-stream
